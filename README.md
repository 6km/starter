# Hono Starter Templates

You can start to build a Hono application with `create hono` command:

```bash
# npm
npm create hono@latest

# yarn
yarn create hono

# pnpm
pnpm create hono@latest

# bun
bun create hono@latest

# deno
deno run -A npm:create-hono@latest
```
